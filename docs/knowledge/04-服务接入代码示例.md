# 服务接入代码示例

> 以下代码为模板示例，具体字段请根据 `sdp service view <service-id>` 返回的 `requestExample` 和 `responseExample` 进行调整。

## 通用说明

所有服务调用都需要：
- **BaseURL**: `https://service.sd.10086.cn`
- **认证方式**: Header 中携带 `token`（通过浏览器插件获取）
- **协议**: 大多数为 HTTP/HTTPS，通常为 POST
- **Content-Type**: `application/json`

---

## Python 示例

```python
import requests
import json

BASE_URL = "https://service.sd.10086.cn"
TOKEN = "你的token"

def call_api(service_path, payload):
    """通用 API 调用封装"""
    url = f"{BASE_URL}{service_path}"
    headers = {
        "Content-Type": "application/json",
        "token": TOKEN
    }
    
    try:
        resp = requests.post(url, headers=headers, json=payload, timeout=30)
        resp.raise_for_status()
        return resp.json()
    except requests.exceptions.RequestException as e:
        print(f"请求失败: {e}")
        return None

# 示例：调用防火墙绑定校验接口
result = call_api(
    "/openapi/GroupServicer?method=checkLineBindFW",
    {
        "subsId": "537*******",
        "region": "537",
        "privId": "pg.vi.cmnet"
    }
)
print(json.dumps(result, indent=2, ensure_ascii=False))
```

## Java 示例（Spring Boot）

```java
import org.springframework.http.*;
import org.springframework.web.client.RestTemplate;
import java.util.*;

public class SdApiClient {
    private static final String BASE_URL = "https://service.sd.10086.cn";
    private static final String TOKEN = "你的token";
    
    public static Map<String, Object> callApi(String path, Map<String, Object> body) {
        RestTemplate restTemplate = new RestTemplate();
        HttpHeaders headers = new HttpHeaders();
        headers.setContentType(MediaType.APPLICATION_JSON);
        headers.set("token", TOKEN);
        
        HttpEntity<Map<String, Object>> entity = new HttpEntity<>(body, headers);
        ResponseEntity<Map> response = restTemplate.exchange(
            BASE_URL + path,
            HttpMethod.POST,
            entity,
            Map.class
        );
        return response.getBody();
    }
}
```

## Go 示例

```go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
    "time"
)

const (
    baseURL = "https://service.sd.10086.cn"
    token   = "你的token"
)

func callAPI(path string, payload map[string]interface{}) (map[string]interface{}, error) {
    url := baseURL + path
    jsonData, _ := json.Marshal(payload)
    
    req, err := http.NewRequest("POST", url, bytes.NewBuffer(jsonData))
    if err != nil {
        return nil, err
    }
    
    req.Header.Set("Content-Type", "application/json")
    req.Header.Set("token", token)
    
    client := &http.Client{Timeout: 30 * time.Second}
    resp, err := client.Do(req)
    if err != nil {
        return nil, err
    }
    defer resp.Body.Close()
    
    var result map[string]interface{}
    if err := json.NewDecoder(resp.Body).Decode(&result); err != nil {
        return nil, err
    }
    return result, nil
}
```

## 最佳实践

1. **统一封装**: 建议对 `token`、超时、重试、日志做统一封装
2. **错误处理**: 除 HTTP 状态码外，还需检查返回 JSON 中的业务状态码
3. **敏感信息**: `token` 不要硬编码，建议通过环境变量或配置中心注入
4. **超时设置**: 建议设置 10-30 秒超时，网络波动时避免长时间挂起
5. **日志记录**: 记录请求路径、响应状态、耗时，便于问题排查
