# 给 AI 助手的提示词模板

> 复制下面这段内容，直接发给你使用的 AI 助手即可。AI 会根据指引完成安装、登录，并帮你操作山东能力开放平台。

---

## 基础版提示词（直接复制使用）

```
我发给你了一个压缩包「sdp-package.zip」，里面包含：
1. 山东能力平台 CLI 工具（6 个平台的二进制文件）
2. Chrome 浏览器扩展文件夹「sdp-login-helper」
3. AI 使用手册「ai-usage-guide.md」

请你：
1. 根据你的系统架构选择合适的二进制文件，安装 CLI 工具（重命名为 sdp 或 sdp.exe 并加入 PATH）
2. 释放浏览器扩展文件「sdp-login-helper」到桌面或某个目录，并告诉我释放路径
3. 我会按照你的指引导入 Chrome 插件并登录平台，然后把登录 Token 发给你
4. 你执行登录后，帮我验证是否连接成功（运行 sdp ability list --size 3）
5. 之后我会根据需求让你帮我查询能力、搜索服务、查看接口详情等

安装过程中如果遇到问题，请直接告诉我详细的解决步骤。
```

## 完整版提示词（含功能预期，推荐）

```
我发给你了一个压缩包「sdp-package.zip」，里面包含山东能力开放平台 CLI 工具。请你按以下步骤操作：

### 第一步：安装 CLI 工具
根据你所在系统的架构，从压缩包里选择对应的二进制文件安装到 PATH 中：
- Windows x64 → sdp-windows-x64.exe
- Windows ARM64 → sdp-windows-arm64.exe
- Linux x64 → sdp-linux-x64
- Linux ARM64 → sdp-linux-arm64
- macOS Intel → sdp-darwin-x64
- macOS Apple Silicon → sdp-darwin-arm64

安装好后运行 `sdp --help` 确认可用。

### 第二步：准备浏览器插件
运行 `sdp helper extract [目录]` 把浏览器扩展「sdp-login-helper」释放到桌面或其他便于我找到的位置，然后告诉我完整路径。

### 第三步：引导我完成登录
我会按以下步骤操作：
1. 打开 Chrome，访问 chrome://extensions/
2. 开启「开发者模式」，加载你释放的「sdp-login-helper」文件夹
3. 访问 https://service.sd.10086.cn/aaas/ 登录平台
4. 点击插件图标，复制弹出的 `sdp login ...` 命令发给你
5. 你执行该命令完成登录

### 第四步：验证连接
登录成功后，请你运行 `sdp ability list --size 3` 验证平台连接是否正常。

### 第五步：你的增值能力
除了基础查询外，希望你还能帮我做这些事：
- 当我不知道选什么能力时，你可以查询平台能力列表后，根据我的业务场景给出能力组合和解决方案建议
- 当我需要对接某个 API 时，你可以查询服务详情，并根据接口定义（URL、请求示例、响应示例）生成对应的调用代码（如 Java/Python/Go/Shell 等）
- 帮我整理能力对比表格、输出接口文档草案、生成项目 README 中的能力依赖说明等

现在请开始安装，完成后告诉我进展。
```

## 按场景细分的追加指令

安装完成后，你可以根据具体需求，复制下面的某一段发给 AI：

### 场景 1：我想做某个业务，不知道用什么能力
```
我想做「智慧景区客流预警」业务，请你：
1. 查询平台上的能力列表
2. 搜索可能相关的关键词（如定位、短信、大数据、客流等）
3. 基于查询结果，给我一份能力选型建议和整体解决方案思路
```

### 场景 2：我想对接某个 API，需要示例代码
```
我想对接「防火墙绑定互联网校验」这个接口（服务ID：SE202604081915467601182242），请你：
1. 查询该服务的完整详情
2. 根据 requestUrl、requestExample、responseExample 生成一份 Java（或 Python/Go）调用示例
3. 补充必要的错误处理说明和依赖清单
```

### 场景 3：帮我整理一份能力清单文档
```
请你查询平台上所有与「短信」相关的能力和服务，整理成一份 Markdown 表格，包含：
- 能力/服务名称
- ID
- 功能简介
- 提供方
- 是否已订购（如有状态字段）
- 推荐的适用场景
```

### 场景 4：帮我做技术预研
```
我正在评估是否使用山东能力平台的「机器视觉」类能力，请你：
1. 搜索并查看机器视觉分类下的所有能力
2. 对每个能力的核心功能、提供方、典型应用场景做简要分析
3. 输出一份技术预研报告，包含能力对比、接入建议、潜在风险提示
```

## 小提示

- 如果 AI 提示"请先登录"，说明你发给它的 Token 已过期，重新点击浏览器插件复制最新的 `sdp login ...` 命令即可。
- 如果某些能力返回"空数组"，通常是因为该能力下没有挂载服务，或者你还没有在网页端完成订购。
- 涉及订购、应用授权等写操作，AI 目前会引导你手动在浏览器完成。
