## 能力详解

### 1. 短信定位
- **技术基础**：消息 + LBS + 地图位置绘制 + H5 交互
- **载体**：支持以不同消息载体发送定位请求
- **输出**：自动获取当前所处位置的定位结果
- **特点**：无需安装 APP，通过短信即可完成定位交互

### 2. 电子围栏
- **核心功能**：
  - 新增电子围栏（划定地理边界）
  - 删除电子围栏
  - 实时监测区域内人口动态
- **数据维度**：区域内人数、人口密度、流动趋势
- **更新频率**：支持实时或准实时监测

## 应用场景

### 场景 A：景区客流管控
- 能力组合：电子围栏 + 大数据短信触达
- 流程：
  1. 划定景区电子围栏
  2. 实时统计围栏内游客数量
  3. 接近或超过承载量时，自动触发短信预警
- 价值：预防踩踏事故，优化游客体验

### 场景 B：应急管理疏散
- 能力组合：短信定位 + 电子围栏
- 流程：
  1. 灾害发生时划定疏散区域电子围栏
  2. 通过短信定位获取被困人员位置
  3. 结合围栏数据指挥精准救援
- 价值：提高救援效率，降低人员伤亡

### 场景 C：智慧商圈运营
- 能力组合：电子围栏 + 大数据短信触达
- 流程：
  1. 在商圈周边设置电子围栏
  2. 分析客流高峰时段和驻留时长
  3. 向围栏内用户推送精准营销短信
- 价值：提升转化率，优化商户布局

## 接入建议

1. **精度说明**：定位精度受基站密度、环境遮挡影响，城区通常可达百米级
2. **合规要求**：位置数据属于敏感个人信息，必须获得用户授权
3. **围栏绘制**：建议提前在测试环境验证围栏边界的准确性
4. **数据更新**：高实时性场景需确认接口的数据刷新频率
