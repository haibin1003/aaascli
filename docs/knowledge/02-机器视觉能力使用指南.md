## 能力详解

### 1. 通用场景目标检测
- **技术基础**：深度学习
- **识别对象**：图像/视频中的常见物体
- **应用领域**：文娱、交通、工业、金融
- **输出结果**：物体的位置坐标（Bounding Box）+ 类别标签

### 2. OCR通用文字识别
- **支持场景**：拍照识别、内容监管、纸质文档电子化
- **识别内容**：中文、英文、数字
- **输出结果**：文字内容 + 文字在图像中的位置

### 3. 凭证识别
- **支持票种**：增值税专票、卷式发票、火车票
- **输出字段**：发票代码、发票号码、金额、日期等关键字段
- **应用**：财务报销自动化、发票验真

### 4. OCR身份证识别
- **输出信息**：姓名、身份证号、出生年月、性别、地址、证件有效期
- **应用场景**：电信实名制、个人征信、身份核验

## 应用场景

### 场景 A：智慧营业厅
- 能力组合：OCR身份证识别 + 凭证识别
- 流程：用户拍摄身份证 → 自动提取信息填充表单 → 拍摄发票自动报销
- 效果：减少人工录入，提升办理效率

### 场景 B：视频监控分析
- 能力组合：通用场景目标检测 + 智能视频内容浓缩
- 流程：监控视频流 → 检测异常物体/行为 → 自动提取关键片段生成告警摘要
- 效果：7x24 小时自动巡检，降低人力成本

### 场景 C：文档数字化
- 能力组合：OCR通用文字识别
- 流程：批量扫描纸质文档 → OCR 提取文字 → 结构化存储
- 效果：档案电子化，支持全文检索

## 接入建议

1. **图像质量**：建议上传分辨率 1080P 以上、光线充足的图像
2. **格式支持**：通常支持 JPG、PNG、BMP 格式
3. **隐私合规**：涉及人脸、身份证等敏感信息时，需符合数据安全法规
4. **并发限制**：大流量场景需提前评估并发能力并申请扩容
