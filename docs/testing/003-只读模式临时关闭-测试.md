## 1. 测试概述

本文档定义只读模式临时关闭功能的测试用例和验收标准。

## 2. 测试用例

### TC1: 临时关闭只读模式 - 成功

| 项目 | 内容 |
|------|------|
| 测试目的 | 验证带 --duration 参数的临时关闭功能 |
| 前置条件 | 当前处于只读模式 |
| 测试步骤 | 1. 执行 `lc readonly off --duration 30m` |
| 预期结果 | 1. 返回成功状态<br>2. readonly=false<br>3. temporary=true<br>4. expireAt 为当前时间+30分钟<br>5. remainingMinutes=30 |

### TC2: 临时关闭 - 查看状态

| 项目 | 内容 |
|------|------|
| 测试目的 | 验证临时关闭期间状态显示正确 |
| 前置条件 | 已执行临时关闭命令（30分钟） |
| 测试步骤 | 1. 执行 `lc readonly` |
| 预期结果 | 1. 显示 readonly=false<br>2. temporary=true<br>3. 显示剩余时间（约30分钟）<br>4. 描述包含"临时读写模式" |

| 项目 | 内容 |
