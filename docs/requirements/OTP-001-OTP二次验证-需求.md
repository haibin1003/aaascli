## 1. 背景与目标

### 1.1 背景
LC CLI（灵畿CLI）是一个用于研发管理的命令行工具，可以执行代码合并、需求管理、任务操作等敏感操作。为防止误操作或未经授权的操作导致数据丢失，需要引入二次验证机制。

### 1.2 目标
- 为危险操作提供 OTP (One-Time Password) 二次验证保护
- 支持标准的 TOTP (Time-based OTP) 协议，兼容主流身份验证器应用
- 提供良好的用户体验，避免频繁输入验证码
- OTP 配置操作本身不受只读模式限制

## 3. 危险操作定义

### 3.1 默认保护列表

系统默认保护以下**不可逆**的高风险操作：

| 操作命令 | 风险描述 | 风险等级 | 实现状态 |
|---------|---------|---------|----------|
| `lc repo delete` | 删除仓库，所有代码和历史记录永久丢失 | Critical | ✅ 默认保护 |
| `lc readonly off` | 关闭只读模式，开放所有写入操作 | Medium | ✅ 默认保护 |

> **原则**: 默认只保护不可逆操作。可逆操作（如 `pr merge` 可通过 git revert 回滚）不列入默认保护，用户可自行添加。

### 3.2 可动态添加的保护操作

用户可通过 `lc otp config add` 添加以下操作到保护列表：

| 操作命令 | 风险描述 | 风险等级 |
|---------|---------|---------|
| `lc pr merge` | 合并代码，可能影响生产环境 | High |
| `lc pr create` | 创建合并请求 | Medium |
| `lc pr review` | 审核合并请求 | Medium |
| `lc req delete` | 删除需求，数据无法恢复 | High |
| `lc task delete` | 删除任务，数据无法恢复 | Medium |
| `lc bug delete` | 删除缺陷，数据无法恢复 | Medium |

## 6. 术语表

| 术语 | 说明 |
|------|------|
| OTP | One-Time Password，一次性密码 |
| TOTP | Time-based One-Time Password，基于时间的一次性密码，RFC 6238 标准 |
| 会话 | 验证成功后的有效时间段（默认 5 分钟） |
| 备用码 | 用于账户恢复的静态密码（10个，8位十六进制） |
| 时间窗口 | TOTP 的时间间隔，默认 30 秒 |
| 灵畿CLI | LC CLI 的中文名称，二维码中显示为 issuer |

## 7. 命令列表

### 7.1 OTP 管理命令

| 命令 | 功能 | 是否需要 OTP 验证 | 是否受只读限制 |
|------|------|-----------------|----------------|
| `lc otp setup` | 初始化 OTP，生成密钥和二维码 | 否 | 否 |
| `lc otp verify [code]` | 验证 OTP 密码，创建会话 | 否 | 否 |
| `lc otp disable` | 关闭 OTP 功能 | **是** | 否 |
| `lc otp status` | 查看 OTP 启用状态和会话状态 | 否 | 否 |

### 7.2 OTP 配置命令（动态保护列表）

| 命令 | 功能 | 是否需要 OTP 验证 | 是否受只读限制 |
|------|------|-----------------|----------------|
| `lc otp config list` | 查看当前保护命令列表 | 否 | 否 |
| `lc otp config add [cmd]` | 添加命令到保护列表 | 否 | 否 |
| `lc otp config remove [cmd]` | 从保护列表移除命令 | 否 | 否 |
| `lc otp config reset` | 重置为默认保护列表 | 否 | 否 |

### 7.3 配置字段

| 配置项 | 类型 | 说明 | 默认值 |
|--------|------|------|--------|
| `otp.enabled` | bool | 是否启用 OTP | false |
| `otp.secret` | string | TOTP 密钥（Base32） | - |
| `otp.verifiedAt` | string | 最后验证时间（ISO8601） | - |
| `otp.sessionExpiryMinutes` | int | 会话有效期（分钟） | 5 |
| `otp.protectedCommands` | []string | 自定义保护命令列表 | 空（使用默认） |
