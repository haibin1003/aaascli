## 1. 设计概述

为只读模式增加临时关闭功能，允许用户通过 `--duration` 参数指定临时关闭时长，到期后自动恢复只读模式。

## 2. 配置结构扩展

### 2.1 Config 结构体修改

在 `internal/config/config.go` 中添加以下字段：

```go
type Config struct {
    Cookie         string            `json:"cookie"`
    Readonly       bool              `json:"readonly"`        // 当前是否只读
    TempOffExpireAt *time.Time       `json:"tempOffExpireAt,omitempty"` // 临时关闭过期时间
    // ... 其他字段
}
```

### 2.2 配置说明

- `TempOffExpireAt`：临时关闭的过期时间，nil 表示非临时关闭
- 当 `Readonly=false` 且 `TempOffExpireAt` 不为 nil 时，表示临时关闭状态
- 当 `Readonly=false` 且 `TempOffExpireAt` 为 nil 时，表示永久关闭

## 3. 核心逻辑设计

### 3.1 临时关闭流程

```
用户执行: lc readonly off --duration 30m

1. 解析 --duration 参数，转换为时间.Duration
2. 计算过期时间: now + duration
3. 更新配置:
   - Readonly = false
   - TempOffExpireAt = expireAt
4. 保存配置
5. 返回临时关闭状态，包含过期时间和剩余分钟数
```

### 3.2 只读检查流程（增强）

```
IsReadonly() 函数增强:

1. 加载配置
2. 如果 Readonly = true，返回 true
3. 如果 Readonly = false:
   a. 检查 TempOffExpireAt 是否为 nil
      - 为 nil: 永久关闭，返回 false
      - 不为 nil: 检查是否过期
        * 未过期: 返回 false（允许写入）
        * 已过期: 自动恢复只读模式，返回 true
```

### 3.3 自动恢复机制

```go
func IsReadonly() bool {
    cfg, err := config.LoadConfigWithDefaults(config.GetDefaultConfigPath())
    if err != nil {
        return true
    }

    // 如果已经是只读模式，直接返回
    if cfg.Readonly {
        return true
    }

    // 检查是否是临时关闭且已过期
    if cfg.TempOffExpireAt != nil && time.Now().After(*cfg.TempOffExpireAt) {
        // 自动恢复只读模式
        cfg.Readonly = true
        cfg.TempOffExpireAt = nil
        config.SaveConfig(cfg)
        return true
    }

    return false
}
```

## 5. 时长解析设计

### 5.1 解析函数

```go
// ParseDuration 解析时长字符串，返回 time.Duration
// 支持格式: 10m, 30m, 1h, 2h
// 限制: 最小 1m，最大 24h
func ParseDuration(s string) (time.Duration, error) {
    if s == "" {
        return 0, fmt.Errorf("时长不能为空")
    }

    // 使用 time.ParseDuration 解析
    d, err := time.ParseDuration(s)
    if err != nil {
        return 0, fmt.Errorf("无效时长格式 %q，支持格式如: 10m, 30m, 1h", s)
    }

    // 验证范围
    if d < time.Minute {
        return 0, fmt.Errorf("时长不能小于 1 分钟")
    }
    if d > 24*time.Hour {
        return 0, fmt.Errorf("时长不能大于 24 小时")
    }

    return d, nil
}
```

## 6. 状态显示增强

### 6.1 GetReadonlyStatus 增强

```go
func GetReadonlyStatus(cfg *config.Config) map[string]interface{} {
    if cfg.Readonly {
        // 原有只读状态返回
        return map[string]interface{}{
            "readonly": true,
            "description": "当前处于只读模式...",
            // ...
        }
    }

    // 非只读模式
    result := map[string]interface{}{
        "readonly": false,
        "description": "当前处于读写模式，可以执行所有操作",
        "warning": "请谨慎执行删除和修改操作...",
    }

    // 添加临时关闭信息
    if cfg.TempOffExpireAt != nil {
        remaining := time.Until(*cfg.TempOffExpireAt)
        if remaining > 0 {
            result["temporary"] = true
            result["expireAt"] = cfg.TempOffExpireAt.Format(time.RFC3339)
            result["remainingMinutes"] = int(remaining.Minutes())
            result["description"] = fmt.Sprintf(
                "当前处于临时读写模式，将在 %d 分钟后自动恢复只读",
                int(remaining.Minutes()))
        }
    }

    return result
}
```

## 7. 错误处理

### 7.1 新增错误类型

```go
// TemporaryOffExpiredError 临时关闭已过期错误
type TemporaryOffExpiredError struct {
    ExpiredAt time.Time
}

func (e *TemporaryOffExpiredError) Error() string {
    return fmt.Sprintf("临时读写模式已过期（%s），已自动恢复只读模式",
        e.ExpiredAt.Format("2006-01-02 15:04:05"))
}
```
