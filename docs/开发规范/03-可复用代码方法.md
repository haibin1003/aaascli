## 命令执行框架

### 标准命令执行模式

所有命令应使用 `common.Execute` 执行，确保统一的输出格式和错误处理：

```go
common.Execute(func(ctx *common.CommandContext) (interface{}, error) {
    // 业务逻辑
    result, err := api.SomeOperation(ctx, ...)
    if err != nil {
        return nil, fmt.Errorf("操作失败: %w", err)
    }
    return result, nil
}, common.ExecuteOptions{
    DebugMode: debugMode,
    Insecure:  insecureSkipVerify,
    DryRun:    dryRunMode,
    Logger:    &logger,
})
```

### 自定义输出

对于需要自定义输出的命令（如非 JSON 输出），使用 `ExecuteWithOutput`：

```go
common.ExecuteWithOutput(func(ctx *common.CommandContext) error {
    // 自行处理输出
    fmt.Println("自定义输出")
    return nil
}, common.ExecuteOptions{
    DebugMode: debugMode,
    Insecure:  insecureSkipVerify,
})
```

## 自动探测

### 获取自动探测上下文

```go
// 尝试自动探测 Git 仓库上下文
autoResult := common.TryAutoDetect(true)  // true = 要求必须在 Git 仓库中
if autoResult.Success {
    workspaceKey = autoResult.Context.WorkspaceKey
    workspaceName = autoResult.Context.WorkspaceName
    gitProjectID = autoResult.Context.GitProjectID
}
```

### 自动探测与参数填充

```go
// 1. 获取显式参数
workspaceKey, _ := cmd.Flags().GetString("workspace-key")

// 2. 如果未指定，尝试自动探测
if workspaceKey == "" {
    autoResult := common.TryAutoDetect(true)
    if autoResult.Success {
        workspaceKey = autoResult.Context.WorkspaceKey
        if workspaceName == "" {
            workspaceName = autoResult.Context.WorkspaceName
        }
    } else {
        common.PrintAutoDetectError(autoResult.Error)
        os.Exit(1)
    }
}
```

### 打印探测信息

```go
// 在调试模式下打印自动探测信息
if debugMode && autoResult.Success {
    common.PrintAutoDetectInfo(autoResult.Context, logger)
}
```

## 错误处理

### 统一错误输出

```go
// 错误时输出统一 JSON 格式
common.PrintError(err)

// 或手动输出错误
fmt.Fprintf(os.Stderr, "错误: %v\n", err)
common.PrintErrorResult(err)
```

### 错误包装

使用 `fmt.Errorf` 的 `%w` 动词包装错误，保留错误链：

```go
result, err := api.CreateRequirement(ctx, req)
if err != nil {
    return nil, fmt.Errorf("创建需求失败: %w", err)
}
```

### 错误分类

```go
type ErrorDetail struct {
    Code       string `json:"code"`
    Message    string `json:"message"`
    Details    string `json:"details,omitempty"`
    Suggestion string `json:"suggestion,omitempty"`

// 使用特定错误码
return nil, &ErrorDetail{
    Code:       "REQUIREMENT_NOT_FOUND",
    Message:    "需求不存在",
```

## JSON 处理

### 统一 JSON 输出

```go
// 打印格式化 JSON
common.PrintJSON(data)

// 手动序列化
output, err := json.MarshalIndent(data, "", "  ")
if err != nil {
    return fmt.Errorf("序列化失败: %w", err)
}
fmt.Println(string(output))
```

### 解析 API 响应

```go
// 解析嵌套 JSON 响应
var response struct {
    Success bool `json:"success"`
    Data    struct {
        Items []struct {
            ObjectID string `json:"objectId"`
            Name     string `json:"name"`
        } `json:"items"`
        Count int `json:"count"`
    } `json:"data"`
}

if err := json.Unmarshal(body, &response); err != nil {
    return fmt.Errorf("解析响应失败: %w", err)
}

// 或使用 map[string]interface{} 进行动态解析
var result map[string]interface{}
if err := json.Unmarshal(body, &result); err != nil {
    return err
}

// 安全地提取嵌套字段
if data, ok := result["data"].(map[string]interface{}); ok {
    if items, ok := data["items"].([]interface{}); ok {
        // 处理 items
    }
}
```

### 从富文本提取纯文本

```go
// internal/common/util.go
func ExtractTextFromRichText(value interface{}) string {
    if value == nil {
        return ""
    }

    // 处理 []interface{} 类型
    if arr, ok := value.([]interface{}); ok {
        var result string
        for _, item := range arr {
            if block, ok := item.(map[string]interface{}); ok {
                if children, ok := block["children"].([]interface{}); ok {
                    for _, child := range children {
                        if childMap, ok := child.(map[string]interface{}); ok {
                            if text, ok := childMap["text"].(string); ok {
                                result += text
                            }
                        }
                    }
                }
            }
            result += "\n"
        }
        return strings.TrimSpace(result)
    }

    // 如果是字符串直接返回
    if s, ok := value.(string); ok {
        return s
    }

    return ""
}
```

## Git 操作

### 探测 Git 仓库信息

```go
// internal/common/autodetect.go

// 检查是否在 Git 仓库中
func isGitRepo(path string) bool {
    cmd := exec.Command("git", "-C", path, "rev-parse", "--git-dir")
    err := cmd.Run()
    return err == nil
}

// 获取仓库名称
func getRepoName(path string) (string, error) {
    cmd := exec.Command("git", "-C", path, "rev-parse", "--show-toplevel")
    output, err := cmd.Output()
    if err != nil {
        return "", err
    }
    return filepath.Base(strings.TrimSpace(string(output))), nil
}

// 获取 remote URL
func getRemoteURL(path string) (string, error) {
    cmd := exec.Command("git", "-C", path, "remote", "get-url", "origin")
    output, err := cmd.Output()
    if err != nil {
        return "", err
    }
    return strings.TrimSpace(string(output)), nil
}
```

### 执行 Git 命令

```go
// e2e/framework/util.go
func ExecGit(t *testing.T, dir string, args ...string) {
    t.Helper()
    cmd := exec.Command("git", args...)
    cmd.Dir = dir
    output, err := cmd.CombinedOutput()
    if err != nil {
        t.Fatalf("git %s failed: %v\nOutput: %s", strings.Join(args, " "), err, output)
    }
}
```

## 用户输入处理

### 从 YAML 文件读取

```go
// internal/common/yaml.go
import "gopkg.in/yaml.v3"

func LoadFromYAMLFile(filename string, v interface{}) error {
    data, err := os.ReadFile(filename)
    if err != nil {
        return fmt.Errorf("读取文件失败: %w", err)
    }
    if err := yaml.Unmarshal(data, v); err != nil {
        return fmt.Errorf("解析 YAML 失败: %w", err)
    }
    return nil
}
```

### 从管道读取输入

```go
// 检查是否有管道输入
func hasPipeInput() bool {
    stat, _ := os.Stdin.Stat()
    return (stat.Mode() & os.ModeCharDevice) == 0
}

// 读取管道输入
func readPipeInput() (string, error) {
    data, err := io.ReadAll(os.Stdin)
    if err != nil {
        return "", err
    }
    return string(data), nil
}
```

### 合并文件和管道输入

```go
// 优先级：显式参数 > 文件 > 管道
func getInputContent(filePath string) (string, error) {
    // 1. 如果有文件，从文件读取
    if filePath != "" {
        data, err := os.ReadFile(filePath)
        if err != nil {
            return "", fmt.Errorf("读取文件失败: %w", err)
        }
        return string(data), nil
    }

    // 2. 检查管道输入
    if hasPipeInput() {
        return readPipeInput()
    }

    return "", nil
}
```

### 重试机制

```go
// e2e/framework/util.go

// Retry 重试执行函数
func Retry(t *testing.T, maxAttempts int, delay time.Duration, fn func() error) error {
    t.Helper()

    var lastErr error
    for i := 0; i < maxAttempts; i++ {
        if err := fn(); err == nil {
            return nil
        } else {
            lastErr = err
            t.Logf("Attempt %d failed: %v, retrying in %v...", i+1, err, delay)
            time.Sleep(delay)
        }
    }
    return fmt.Errorf("failed after %d attempts: %w", maxAttempts, lastErr)
}

// PollForCondition 轮询直到条件满足或超时
func PollForCondition(t *testing.T, maxRetries int, delay time.Duration, condition func() bool) bool {
    t.Helper()
    for i := 0; i < maxRetries; i++ {
        if condition() {
            return true
        }
        t.Logf("Polling attempt %d/%d failed, waiting...", i+1, maxRetries)
        time.Sleep(delay)
    }
    return false
}
```

### 文件操作

```go
// e2e/framework/util.go

// CreateTempFile 创建临时文件并返回路径
func CreateTempFile(t *testing.T, prefix string, content string) string {
    t.Helper()
    tempFile, err := os.CreateTemp("", prefix+"-*.yaml")
    if err != nil {
        t.Fatalf("Failed to create temp file: %v", err)
    }
    defer tempFile.Close()

    if _, err := tempFile.WriteString(content); err != nil {
        os.Remove(tempFile.Name())
        t.Fatalf("Failed to write temp file: %v", err)
    }

    // 注册清理
    t.Cleanup(func() {
        os.Remove(tempFile.Name())
    })

    return tempFile.Name()
}

// CreateTempDir 创建临时目录
func CreateTempDir(t *testing.T) string {
    t.Helper()
    tempDir, err := os.MkdirTemp("", "lc-e2e-test-*")
    if err != nil {
        t.Fatalf("Failed to create temp dir: %v", err)
    }
    return tempDir
}
```

```go
func URLEncode(s string) string {
    s = strings.ReplaceAll(s, "@", "%40")
    s = strings.ReplaceAll(s, "+", "%2B")
```

## 设计模式

### 1. 命令模板模式

```go
// 所有命令遵循统一的执行模板
var someCmd = &cobra.Command{
    Run: func(cmd *cobra.Command, args []string) {
        // 1. 获取参数
        param := getParam(cmd, args)

        // 2. 尝试自动探测
        if param == "" {
            param = autoDetect()
        }

        // 3. 验证参数
        if param == "" {
            returnError("缺少必需参数")
            return
        }

        // 4. 执行
        common.Execute(func(ctx *common.CommandContext) (interface{}, error) {
            return doSomething(ctx, param)
        }, opts)
    },
}
```

### 2. 资源清理模式

```go
// 使用 defer 确保资源清理
func doSomething() error {
    resource, err := acquireResource()
    if err != nil {
        return err
    }
    defer releaseResource(resource)  // 确保释放

    // 使用 resource...
    return nil
}
```

### 3. 缓存模式

```go
// 自动探测结果缓存
var (
    cachedDetectResult *AutoDetectContext
    cachedDetectMu     sync.RWMutex
    cacheValidDuration = 5 * time.Minute
)

func GetAutoDetectContext() (*AutoDetectContext, error) {
    // 1. 先检查缓存
    cachedDetectMu.RLock()
    if cachedDetectResult != nil && time.Since(cachedDetectResult.DetectTime) < cacheValidDuration {
        result := cachedDetectResult
        cachedDetectMu.RUnlock()
        return result, nil
    }
    cachedDetectMu.RUnlock()

    // 2. 执行探测
    ctx, err := performDetect()
    if err != nil {
        return nil, err
    }

    // 3. 更新缓存
    cachedDetectMu.Lock()
    cachedDetectResult = ctx
    cachedDetectMu.Unlock()

    return ctx, nil
}
```

## 最佳实践

1. **优先使用自动探测**：减少用户输入，提高易用性
2. **统一错误处理**：使用 `common.Execute` 确保输出格式一致
3. **参数验证前置**：在 `Execute` 回调外验证参数，提前返回错误
4. **资源及时释放**：使用 `defer` 确保资源清理
5. **日志分级输出**：使用 `ctx.Logger.Debug()` 记录调试信息
6. **并发安全**：使用 `sync.RWMutex` 保护共享状态
