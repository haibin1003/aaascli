## 典型服务接口

### 1. 自定义短信下发
- **服务ID**: `SE202402071658102321000830`
- **接口编码**: `sendSMSInfo`
- **适用场景**: 需要自定义内容的企业短信下发

### 2. 下发机主授权影像认证上传短信
- **服务ID**: `SE202510221644582581177937`
- **接口编码**: `BLCSSendSMSForManualAuth`
- **适用场景**: 实名认证、业务办理时的短信确认

### 3. 前台短信发送
- **服务ID**: `SE202508201045179361024513`
- **接口编码**: `sendSmsForEbus`
- **适用场景**: 业务前台系统短信发送

## 应用场景

### 场景 A：政务通知
- 能力组合："云信"位置短信触达能力
- 流程：划定地理围栏 → 筛选区域内移动用户 → 批量发送政务通知短信
- 优势：支持公益宣传、政策通知、应急管理等场景

### 场景 B：企业营销
- 能力组合：大数据短信触达能力
- 流程：用户画像分析 → 精准分群 → 动态内容发送
- 优势：解决传统短信"广撒网"效率低的问题

### 场景 C：业务办理确认
- 能力组合：短信能力 + 短信二次确认接口
- 流程：业务发起 → 发送验证码/确认短信 → 用户回复上行短信确认
- 适用：套餐变更、业务订购等需用户确认的场景

## 接入建议

1. **白名单要求**：党政军事业单位、国有企业、水电煤等优先开通
2. **内容审核**：短信内容需符合运营商规范，不得含违规内容
3. **签名要求**：通常需要配置企业签名（如【中国移动】）
4. **频次控制**：注意设置合理的发送频次，避免用户投诉
